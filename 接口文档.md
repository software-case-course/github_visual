# 1.仓库

## 1.1根据语言、年份获得仓库数
- API地址：/CheckLanguageUseByYear
- 请求样例：http://localhost:8081/GithubVisualize/CheckLanguageUseByYear?language=java&year=2017
- 必要参数：
- 请求返回值：
    -   请求成功：
        
        >        {
"status": 200,
"msg": "OK",
"data": {
"data": 1504425,
"num": 1,
"info": null
},
"error": null
}

    -   请求失败：
        >

- API地址：/CheckLanguageUseByYear
- 请求样例：http://localhost:8081/GithubVisualize/CheckLanguageUseByYear
- 必要参数：
- 请求返回值：
    -   请求成功：
        
        >     {
					"status": 200,
					"msg": "OK",
					"data": {
					"data": 1504425,
					"num": 1,
					"info": null
					},
					"error": null
					}
		>		}

## 1.2根据语言、年份、月份获得仓库数
- API地址：/CheckLanguageUseByMonth
- 请求样例：http://localhost:8081/GithubVisualize/CheckLanguageUseByMonth?year=2017
- 必要参数：year:年份
- 请求返回值：
    -   请求成功：

"status": 200,
